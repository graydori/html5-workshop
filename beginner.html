HTML: HyperText Markup Language

What does it mean?

It's basically a language to describe (or markup) text. How do you think your browser knows how to layout things on a page? Or how does it know what is bold, what is the title, the paragraphs?

For example To describe something as a paragraph with HTML you would wrap it in a beginning ≮p≯ and ending ≮/p≯. Like so..

HTML consists of elements denoted with angle brackets called tags

≮p≯ is a tag

Remember that HTML is not a programming language, it is a markup language (Browsers read the HTML documents and interpret the tags as the content of the web page)

What makes up a HTML document or Web page?

Anatomy of an HTML Web Page
  • ≮html≯ HTML tags tell the browser where the HTML starts and stops. Every HTML file must begin and end with the html tags.
  • ≮head≯ Describes the HTML page. This element will contain extra information like the title and description of the document
  • ≮body≯ Contents of the HTML page. This element will contain the actual content of the document.
  • and many many more!

Selecting the Markup

We've now organized text in a very standard way. With the power of CSS we can select any or the elements on an HTML Page to force our "creative direction" upon

Select a tag

Let us first select a tag such as : h1, h2, a, p or many more.

Notice! new HTML element style. With this element we specify CSS on a HTML page.

CSS has a standard format starting with a selector of HTML elements then curly brackets wrapped around look and feel specification for those elements. These are called CSS Rules. You can have several rules in a CSS document.

Select elements inside another element

Nesting elements in HTML is very common as pages can get very big. If you want to select a child elements inside a parent element then, use the parent tag followed by a space then the child tag for the CSS selection.

Like so..

Using multiple CSS Selections for one CSS Rule

Repeating yourself is bad - you don't want to repeat look and feel specifications for certain CSS selections. So - to use multiple CSS Selections for one rule, simply separate the selections with commas.

Like so..

I need to be more specific!

Now - if you need to be more specific for certain elements on a page you can add markup to those HTML elements.

  • HTML Using the class attribute on any HTML element you can select elements with specific classes in their class attribute. The class attribute can be used to specify several class names separated by spaces.
  • CSS Selecting them in css is achieved with a period after the tag name followed by the class name.

Style ONE HTML element

There are many times when you only want to select one HTML element on a page.

  • HTML Add the id attribute to the html element and specify a unique name to the element.
  • CSS Use the tag name followed by the number symbol (#) then by the unique id of the element.
  • Note! Each id must be unique on an HTML Page. You can't have two elements with the same id on a page

Sizing

Before we go into sizing of HTML elements we'll need to talk about a unit of measurement to specify the size of an html element

Pixels

Pixels is a unit of measurement relative to your screen. This unit is commonly use in web design.

Look below to see boxes in incrementing pixel sizes using the width and height css properties which take a unit of measurement as their value.

Spacing

margin is a property to specify the spacing around the HTML element.

While, padding is a property to specify the spacing of the text inside the HTML element.

Both of these properties take a unit of measurement as the value. These properties can also take a list of units to specify specific padding and margins for the different sides of the element in the following order : first top, right, bottom, then left separated by spaced

Note! Some elements like p, h1 already have margins and padding from the browser.

Typography

font-family

The font-family property specifies the font of the HTML element.

The value for the font-family property can be any of the fonts on your Computer.

  • Arial
  • Impact
  • Comic Sans MS

font-size

The font-size property specifies the font size of the text in the HTML element.

The value for the font-size property is a unit of measurement.
