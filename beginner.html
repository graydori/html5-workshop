HTML: HyperText Markup Language

What does it mean?

It's basically a language to describe (or markup) text. How do you thing your browser knows how to layout things on a page? Or how does it know what is bold, what is the title, the paragraphs?

For example To describe something is a paragraph with HTML you would wrap it in a beginning ≮p≯ and ending ≮/p≯. Like so.. 

≮p≯
    This is a paragraph
≮p≯

HTML consists of elements denoted with angle brackets called tags

≮p≯ is a tag

Remember that HTML is not a programming language, it is a markup language (Browsers read the HTML documents and interpret the tags as the content of the web page)

What makes up a HTML document or Web page?

≮html≯
    ≮head≯
        ≮title≯Title of the Document≮/title≯
    ≮/head≯
    ≮body≯
        Contents of the Document
    ≮/body≯
≮/html≯
Anatomy of an HTML Web Page
  • ≮html≯ HTML tags tell the browser where the HTML starts and stops. Every HTML file must begin and end with the html tags.
  • ≮head≯ Describes the HTML page. This element will contain extra information like the title and description of the document
  • ≮body≯ Contents of the HTML page. This element will contain the actual content of the document.
  • and many many more!

Linking Pages

a

The "a" tag stands for anchor. It anchors some text on the page to a URL (or another page). This is how the web is connected. A link or anchor on one page to another.

  • Text for the link within the begin and end tag of the anchor element
  • Link to the page using html attributes, specify the link in the anchor element in the "href" attribute of begin tag. URLs are in the form of http://www.blah.com/

CSS: Cascading Style Sheet

  • Style Sheet Document to control the look an feel of HTML
  • Cascading Supports several levels of stylesheets that may override/join with other style sheets
CSS is powerful. It doesn't care about the content or workings of the html but a focus on the look and feel of the HTML.
This is a powerful concept: Separation of Concerns.

The Power of CSS

One HTML Page - so many different ways to show the same content using different CSS

Selecting the Markup (HTML)
