What is HTML?

Html stands for ..

HyperText Markup Language

What does that mean? It's basically a language to describe text

For example To descbibe something is a paragraph with Html you would wrap it in a beggining p and ending p tag. Like so.. 

≮p≯
    This is a paragraph
≮p≯
